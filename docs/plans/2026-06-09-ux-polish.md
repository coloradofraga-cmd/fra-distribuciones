**Files:**
- Modify: `app/carrito/page.tsx`

**Step 1: Importar toast**

```tsx

```

**Step 2: Envolver removeItem con toast**

Buscar donde se llama `removeItem(product.id)` y agregar:

```tsx

toast("Producto eliminado", { duration: 1500 });
```

**Step 3: Toast en updateQuantity → 0**

Si el usuario baja cantidad a 0 y se elimina el ítem, también mostrar toast. El store ya llama `removeItem` internamente cuando `quantity < 1`, así que agregar toast en el handler del botón `-` cuando `quantity === 1`:

```tsx
if (quantity === 1) {
  removeItem(product.id);
  toast("Producto eliminado", { duration: 1500 });
} else {
  updateQuantity(product.id, quantity - 1);
}
```

**Step 4: Verificar**

Ir a /carrito → eliminar producto → toast aparece. Bajar cantidad a 0 → toast aparece.

**Step 1: Crear Skeleton base**

`components/ui/Skeleton.tsx`:

```tsx
import { cn } from "@/lib/utils"; // si no existe, usar className directamente

interface SkeletonProps {
  className?: string;
}

export function Skeleton({ className }: SkeletonProps) {
  return (
    <div
      className={`animate-pulse rounded-lg bg-[var(--color-surface-container-high)] ${className ?? ""}`}
    />
  );
}
```

**Step 2: Crear ProductSkeleton**

`components/ui/ProductSkeleton.tsx`:

```tsx
import { Skeleton } from "./Skeleton";

export function ProductSkeleton() {
  return (
    <div className="flex flex-col gap-2">
      <Skeleton className="w-full aspect-square rounded-2xl" />
      <Skeleton className="h-4 w-3/4" />
      <Skeleton className="h-4 w-1/2" />
      <Skeleton className="h-8 w-full rounded-xl" />
    </div>
  );
}

export function ProductSkeletonGrid({ count = 6 }: { count?: number }) {
  return (
    <div className="grid grid-cols-2 gap-3 px-4">
      {Array.from({ length: count }).map((_, i) => (
        <ProductSkeleton key={i} />
      ))}
    </div>
  );
}
```
